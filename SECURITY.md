# Security Policy

## Supported Versions

All versions of the recipe database are supported. There are no release
channels — the `main` branch is the single source of truth.

## Reporting a Vulnerability

This repository contains coffee recipe data in YAML format. It does not
execute code or handle user data. However, if you discover:

- Malicious or harmful content in a recipe file
- A security issue in the CI pipeline or tooling

Please open a [GitHub Security Advisory](https://github.com/HotThoughts/brew-recipes/security/advisories/new).

You should expect an acknowledgment within 48 hours and an initial triage
within 5 business days.
